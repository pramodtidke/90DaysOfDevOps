# Day 03 – Linux Commands Practice

## Task: Completed

- PROCESS MANAGEMENT:

1. $ ps aux : show all running processes

2. $ top / htop : real time cpu and memory usage

3. pgrep <name> : find the PID a process by name

4. kill <PID> : Gracefully stop the process

5. Kill -9 <PID> : forcefully terminate the process

6. systemctl status <service> : show service health

7. systemctl start/stop/restart <service> : control service

8. jobs : list background jobs

9. command& : runs command in background

- FILE SYSTEM

10. ls : list files and directories

11. cd <dir> : change directory

12. touch pramod.txt : create a empty file

13. mkdir <dir> : create an empty directory

14. cp <src> <dest> : copy from source to destination

15. mv <src> <dest> : move from souce to destination

16. rm file : remove file or delete file

17. cat pramod.txt : view file content

18. chmod 755 pramod.txt : change the file permisson

19. df -h : show disk usage

Networking Troubleshooting

20. ip a : show ip address

21. ping host : checks connectivity

22. curl <url> : test http response

23. nslookup <domain> : check dns resolutions

24. traceroute host : show network path

25. systemctl restart <networkmanager> : restart network service

26. dig <domain> : DNS lookup and troubleshooting
